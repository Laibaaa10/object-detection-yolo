# Real-Time Object Detection System
> AI Course Project — Built with YOLOv8 + OpenCV + Streamlit

## Features
- Real-time object detection using YOLOv8
- Object tracking with unique IDs (ByteTrack)
- Line-crossing counter
- Class filtering (detect only what you need)
- Alert system for specific objects
- Streamlit GUI dashboard
- Supports webcam, image, and video input

## Setup
pip install -r requirements.txt

## Run GUI
streamlit run app.py

## Run Terminal
python main.py

## Project Structure
yolo_detection/
├── detector.py      # YOLO inference + drawing
├── tracker.py       # Line crossing counter
├── app.py           # Streamlit GUI
├── main.py          # Terminal entry point
└── output/          # Saved results
